# Tip Calculator

A simple tip calculator built with React.  
This app allows users to enter a bill amount, select service satisfaction levels for two people, and automatically calculate the tip and total amount to pay.

## Overview

This project was built as a small React practice app.  
The main goal was to practice core React concepts such as state management with `useState`, handling form inputs, conditional rendering, and passing data through props.

## Features

- Enter bill amount
- Select your service satisfaction
- Select your friend's service satisfaction
- Automatically calculate tip amount and total payment
- Reset all inputs with one button

## Tech Stack

- React
- JavaScript
- JSX
- CSS

## What I Learned

Through this project, I practiced:

- Managing state with `useState`
- Building controlled components
- Using props and `children`
- Handling events with `onChange` and `onClick`
- Conditional rendering
- Splitting UI into reusable components

## Note

This project was built as part of a **Challenge exercise from Jonas Schmedtmann's React course on Udemy**.  
It is not a fully original product idea, but a learning project that I implemented myself to strengthen my understanding of React fundamentals.

## Possible Improvements

- Add input validation
- Improve the UI design
- Support currency formatting
- Optimize for mobile devices

## Getting Started

```bash
npm install
npm run dev
```
